#  Portfolio | Desenvolvedora Backend

## 🧠 Tecnologias

- React  
- JavaScript  
- HTML / CSS  
- Consumo de APIs REST  
- Deploy com GitHub Pages

## ⚙️ Funcionalidades

- Navegação simples
